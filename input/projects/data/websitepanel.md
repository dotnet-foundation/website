# Website Panel

[Website Panel](https://www.websitepanel.net/) is a complete portal for Cloud Computing Companies and IT Providers to automate the provisioning of a full suite of services on Windows servers. The powerful, flexible open source WebsitePanel platform gives users simple point-and-click control over Windows Server applications including IIS, SQL Server, Exchange 2010, Sharepoint 2010, OCS, Lync and Hyper-V Deployments.
